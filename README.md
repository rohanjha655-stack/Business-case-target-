# Rohan Jha — Data Analytics Portfolio

Data Analyst with hands-on experience in Python, SQL, and Statistics across e-commerce, logistics, retail, sports, and entertainment domains. Each project below includes the full analysis, statistical testing, and business recommendations.

## Projects

### 📦 [Target Brazil E-Commerce Analysis](./Target-Business_case.pdf)
SQL analysis of 99,441 orders (2016–2018) using BigQuery. Found 136.98% YoY order value growth, identified SP as the top-performing state with the lowest freight cost, and flagged a 6.8% late-delivery rate.
**Tools:** SQL, BigQuery

### 🚚 [Delhivery Logistics & Supply Chain Analytics](./DELHIVERY%20CASE%20STUDY.pdf)
Processed 144K+ shipment records into trip-level data. Used Wilcoxon testing to prove the routing engine (OSRM) significantly underestimates real delivery time, and identified Maharashtra, Gujarat, and Karnataka as the top volume states.
**Tools:** Python, Pandas, SciPy, Scikit-Learn

### 🏏 [IPL Match Strategy & Performance Analysis](./IPL_Match_Strategy_Analysis.pdf)
Analyzed 636 matches and 150K+ deliveries (2008–2017). Used a Chi-Square test to statistically confirm that fielding first after winning the toss improves win rate (55.4% vs. 45.4%, p=0.016).
**Tools:** Python, Pandas, SciPy, Matplotlib, Seaborn

### 🛒 [Walmart Black Friday Purchase Behavior Analysis](./walmart%20case%20study%202.pdf)
Analyzed 550K+ transactions. Used confidence intervals to prove gender is a statistically significant spending driver, while marital status is not — separating real signal from assumption.
**Tools:** Python, Pandas, NumPy, SciPy

### 🎬 [Netflix Content Strategy Analysis](./Netflix_Business_Case_Final1%20(1).pdf)
Analyzed 10,000+ titles to uncover content, genre, and regional consumption trends across 6 years, and identified high-growth international markets.
**Tools:** Python, Pandas, Matplotlib, Seaborn

### 🚲 [Yulu Demand Analysis & Hypothesis Testing](./Yulu.ipynb%20-%20Colab.pdf)
Applied T-Test, Chi-Square, Kruskal-Wallis, and Shapiro-Wilk tests on 10K+ bike-sharing records to determine what actually drives electric cycle demand.
**Tools:** Python, Pandas, SciPy

### 🏋️ [AeroFit Customer Segmentation Analysis](./Aerofit_BusinessCase_Final-%20Colab.pdf)
Customer profiling using contingency tables and conditional/marginal probability analysis to improve product recommendations.
**Tools:** Python, Pandas, Matplotlib, Seaborn
