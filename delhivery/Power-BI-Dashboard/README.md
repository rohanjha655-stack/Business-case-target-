

# 🚚 Delhivery Logistics Analytics Dashboard

## Dashboard Preview

![Dashboard](Dashboard.png)

## Overview
This Power BI dashboard analyzes Delhivery logistics operations using 145K+ trip records.

## KPIs
- Total Trips
- Average Actual Time
- Average Actual Distance
- Average Speed
- Delay Time
- Total Source Centers
- Total Destination Centers

## Visualizations
- Trips Over Time
- Top 10 Source Centers
- Top 10 Destination Centers
- Route Type Distribution
- Top 10 Fastest Source Centers
- Top 10 Source Centers by Average Delay

## Tools Used
- Power BI
- DAX
- Power Query
